# Problem Set 03: Modifiers and Non-Destructive Workflow

### **Goal**
Learn to use modifiers to automate modeling tasks without permanently changing geometry.

### **Tasks**
1. **Subdivision Surface**: Add a `Subdivision Surface` modifier to a cube to make it a sphere.
2. **Mirror Modifier**: Create half of a character's head and use the `Mirror` modifier to complete it.
3. **Array Modifier**: Use the `Array` modifier to create a spiral staircase.
4. **Boolean Modifier**: Use one object to "cut" a hole into another.

### **Practice Drill**
Create a detailed "Cog" or "Gear" using a Cylinder, the `Array` modifier for the teeth, and a `Boolean` modifier to create the center hole.
