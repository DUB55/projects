# Problem Set 06: Regression Models

### **Goal**
Predict continuous numerical values.

### **Tasks**
1. **Linear Regression**: Explain $y = mx + b$ in the context of data.
2. **Multiple Regression**: Predict a value using multiple input features.
3. **Evaluation Metrics**: Calculate R-squared, MAE (Mean Absolute Error), and RMSE.
4. **Overfitting**: Explain how a model can "memorize" data instead of learning patterns.

### **Practice Drill**
Build a model to predict "House Prices" based on square footage and number of bedrooms.
