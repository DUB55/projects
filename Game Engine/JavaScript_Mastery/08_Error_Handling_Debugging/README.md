# Problem Set 08: Error Handling and Debugging

### **Goal**
Find and fix bugs efficiently.

### **Tasks**
1. **Try/Catch/Finally**: Wrap risky code in a try block.
2. **Throwing Errors**: Create a custom error message using `throw new Error(...)`.
3. **Debugger**: Use `debugger;` and browser dev tools to step through code.
4. **Console Methods**: Use `console.table()`, `console.time()`, and `console.trace()`.

### **Practice Drill**
Write a script that intentionally fails and uses error handling to log the error and keep the program running.
