# Figma Mastery

Welcome to the **Figma Mastery** curriculum. This course will take you from basic shapes to building full-scale design systems and high-fidelity prototypes.

## Curriculum Overview
1.  **Frames and Layers**: The foundation of the Figma workspace.
2.  **Vector Networks and Shapes**: Drawing and combining vectors.
3.  **Auto Layout Fundamentals**: Creating responsive, growing components.
4.  **Components and Variants**: Building reusable design elements.
5.  **Styles and Variables**: Managing colors, typography, and data.
6.  **Prototyping Interactions**: Making your designs interactive.
7.  **Design Systems Architecture**: Organizing assets for scale.
8.  **Responsive Grid Systems**: Designing for multiple screen sizes.
9.  **Plugins and Developer Handoff**: Extending Figma and working with engineers.
10. **Advanced Micro-Animations**: Using Smart Animate for high-end UI motion.

## How to use this
Each folder contains a `README.md` with a specific problem set. Complete them in order.
If you get stuck, refer to the `HELP.md` in this directory.
