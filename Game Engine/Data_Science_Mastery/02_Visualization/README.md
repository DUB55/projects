# Data Visualization

Telling stories with data.

## Concepts
- **Matplotlib**: The foundation for plotting in Python.
- **Seaborn**: A high-level library for beautiful statistical graphics.
- **Types of Plots**: Bar charts, Line graphs, Scatter plots, Histograms.
- **Customizing Plots**: Adding titles, labels, and legends.

## Challenge
1. Create a scatter plot showing the relationship between two variables in your dataset.
2. Build a histogram to see the distribution of a single variable.
3. Research what makes a "Good Visualization" (e.g., color choice, clarity).
