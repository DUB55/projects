# Problem Set 08: Expression Basics

### **Goal**
Use simple code to automate animations.

### **Tasks**
1. **Wiggle**: Alt-click a stopwatch and type `wiggle(5, 50)` (frequency, amplitude).
2. **Time**: Use `time * 100` to create constant rotation or movement.
3. **Looping**: Use `loopOut()` to repeat keyframes forever.
4. **Pick Whip**: Use the expression pick whip to link one property to another (e.g., the scale of a circle to the volume of an audio file).

### **Practice Drill**
Create a "Handheld Camera" feel. Apply a `wiggle` expression to the position and rotation of your camera layer.
