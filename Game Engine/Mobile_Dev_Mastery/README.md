# Mobile Development Mastery

Learn how to build and port applications to mobile platforms (iOS and Android).

## Curriculum
1. [01_Cross_Platform_React_Flutter](./01_Cross_Platform_React_Flutter/README.md) - One codebase, two platforms.
2. [02_Native_Android_Kotlin](./02_Native_Android_Kotlin/README.md) - Deep dive into Android internals.
3. [03_Native_iOS_Swift](./03_Native_iOS_Swift/README.md) - Deep dive into Apple's ecosystem.
4. [04_App_Store_Deployment](./04_App_Store_Deployment/README.md) - Packaging, signing, and releasing.

## Key Concept: Porting
- **Web to Mobile**: Using Capacitor or React Native.
- **Desktop to Mobile**: Adapting UI for touch and smaller screens.
- **Games to Mobile**: Optimizing performance for ARM processors.
