# Problem Set 05: Effects and Presets

### **Goal**
Enhance your visuals with built-in effects.

### **Tasks**
1. **Effects Control Panel**: Learn where to adjust effect parameters.
2. **Color Correction**: Use `Lumetri Color` or `Curves` to change a layer's look.
3. **Distortion**: Experiment with `Turbulent Displace` or `Liquify`.
4. **Generating**: Use `Fractal Noise` to create textures like clouds or fire.

### **Practice Drill**
Create a "Digital Glitch" effect. Apply `Fractal Noise` to an Adjustment Layer, animate the `Evolution`, and use it as a displacement map for your footage.
