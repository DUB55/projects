# Native Android Development (Kotlin)

Building directly for the Android ecosystem.

## Concepts
- **Kotlin**: The modern, safe language for Android.
- **Activities and Fragments**: The building blocks of an Android UI.
- **Jetpack Compose**: The modern declarative UI toolkit.
- **Android Manifest**: The "Instruction Manual" for your app.

## Challenge
1. Create a "To-Do List" app using Jetpack Compose.
2. Implement a "Room Database" to save the user's tasks locally.
3. Research what "ProGuard" and "R8" are for Android app security and optimization.
