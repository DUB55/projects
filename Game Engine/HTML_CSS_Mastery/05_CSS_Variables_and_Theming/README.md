# Problem Set 05: CSS Variables and Theming

### **Goal**
Create dynamic, reusable styles using CSS Custom Properties.

### **Tasks**
1. **Declare Variables**: Create a `:root` selector and define `--primary-color`, `--secondary-color`, and `--font-main`.
2. **Usage**: Apply these variables to multiple elements.
3. **Dark Mode**: Create a `.dark-theme` class that overrides the variables.
4. **JavaScript Interaction**: Use JS to change the value of a CSS variable at runtime (e.g., `setProperty('--main-bg', 'blue')`).

### **Practice Drill**
Build a simple landing page header that switches between Light and Dark mode using a button and CSS variables.
