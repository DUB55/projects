# Cloud Computing (AWS/Azure/GCP)

Scaling to millions of users without buying a single server.

## Services to Know
- **Compute**: EC2 (Virtual Servers), Lambda (Serverless functions).
- **Storage**: S3 (Object storage for files).
- **Networking**: VPC (Your private slice of the cloud).

## Challenge
1. Sign up for a "Free Tier" account on AWS or Azure.
2. Launch a small Linux virtual machine (EC2 instance) and connect to it via SSH.
