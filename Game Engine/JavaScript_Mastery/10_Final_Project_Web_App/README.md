# Problem Set 10: Final Project - Web App

### **Goal**
Build a complete, functional web application.

### **Tasks**
1. **Architecture**: Design the state management and component structure.
2. **API Integration**: Fetch data from an external source.
3. **UI/UX**: Create a clean, responsive interface using CSS.
4. **Persistence**: Use `localStorage` to save user data between sessions.

### **Practice Drill**
Build a "Movie Search App" where users can search for movies, see details, and save their favorites to a list.
