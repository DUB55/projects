# Problem Set 06: Monitoring and Logging

### **Goal**
Gain visibility into your system's health and performance.

### **Tasks**
1. **Metrics vs Logs**: Explain the difference between "What happened?" and "How many times?".
2. **Prometheus & Grafana**: Set up a dashboard to visualize CPU and Memory usage.
3. **ELK Stack**: Explain the roles of Elasticsearch, Logstash, and Kibana.
4. **Alerting**: Configure an alert that triggers when a service goes down.

### **Practice Drill**
Instrument a simple Python script to export custom metrics to Prometheus using the `prometheus_client` library.
