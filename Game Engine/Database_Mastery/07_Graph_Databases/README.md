# Problem Set 07: Graph Databases

### **Goal**
Model and query highly connected data.

### **Tasks**
1. **Nodes and Edges**: Explain how graph databases (like Neo4j) differ from relational ones.
2. **Cypher Query Language**: Write a query to find "Friends of Friends" in a social network.
3. **Use Cases**: Why are graph databases better for recommendation engines or fraud detection?
4. **Traversal**: Explain the concept of "Breadth-First Search" (BFS) in a graph.

### **Practice Drill**
Model a "Movie Recommendation" system where Users like Movies, and Movies have Genres.
