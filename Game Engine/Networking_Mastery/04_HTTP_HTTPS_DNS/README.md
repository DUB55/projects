# Problem Set 04: HTTP, HTTPS & DNS

### **Goal**
Understand the protocols that power the World Wide Web.

### **Tasks**
1. **HTTP Methods**: Explain `GET`, `POST`, `PUT`, `DELETE`.
2. **Status Codes**: What do `200`, `404`, `500`, and `301` mean?
3. **DNS Lookup**: Explain the process of turning `google.com` into an IP address.
4. **HTTPS**: Understand how SSL/TLS certificates provide encryption.

### **Practice Drill**
Use `curl -v google.com` in your terminal and analyze the HTTP headers sent and received.
