# Graphics API Mastery

The art and science of rendering pixels on the screen.

## Why Graphics APIs?
- **Performance**: Talking directly to the GPU for maximum speed.
- **Visuals**: Creating realistic lighting, shadows, and effects.
- **Cross-Platform**: Writing code that runs on Windows, Mac, Linux, and Mobile.

## Curriculum
1. [01_OpenGL_Basics](./01_OpenGL_Basics/README.md) - The foundation of modern graphics.
2. [02_Shaders_GLSL](./02_Shaders_GLSL/README.md) - Programming the GPU directly.
