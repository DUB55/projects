# Problem Set 09: Template Metaprogramming

### **Goal**
Perform calculations and generate code at compile-time.

### **Tasks**
1. **SFINAE**: Understand "Substitution Failure Is Not An Error" and its use in template specialization.
2. **Type Traits**: Use `std::is_integral` or `std::is_floating_point` to constrain template types.
3. **Variadic Templates**: Write a template function that can take an arbitrary number of arguments (like a custom `print` function).
4. **constexpr**: Use `constexpr` to ensure a function is evaluated at compile-time.

### **Practice Drill**
Write a compile-time Factorial calculator using template recursion or `constexpr`.
