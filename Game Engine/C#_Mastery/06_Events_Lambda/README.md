# Problem Set 06: Events and Lambda Expressions

### **Goal**
Understand the observer pattern and concise function syntax in C#.

### **Tasks**
1. **Events**: Declare an `event` using the `EventHandler` delegate.
2. **Lambda Expressions**: Rewrite a simple method using `(x) => x * x` syntax.
3. **LINQ Prep**: Use `.Where()` and `.Select()` with lambdas on a `List<int>`.
4. **Anonymous Types**: Create an object on the fly using `var person = new { Name = "Alice", Age = 30 };`.

### **Practice Drill**
Create a "Button" class with a `Clicked` event. Subscribe to it in your main program.
