# Problem Set 06: Nodes and Shaders

### **Goal**
Create complex, realistic materials using the Shader Node editor.

### **Tasks**
1. **Principled BSDF**: Understand the core shader nodes (Base Color, Roughness, Metallic, Normal).
2. **Mixing**: Use a `Mix Shader` node to combine two different materials.
3. **Procedural Textures**: Use `Noise Texture` and `Musgrave Texture` nodes to create organic patterns.
4. **Bump & Normal Maps**: Connect textures to the `Height` or `Normal` inputs to create surface detail.

### **Practice Drill**
Create a "Rusty Metal" material. Use a noise texture to drive a color ramp (between metal and rust colors) and another noise texture for the roughness.
