{{ project_name }} Local Setup
Download, unzip, install, run. Share over LAN with a short PIN.
Download ZIP
Setup Steps
  1. Unzip the downloaded archive.
  2. Open a terminal in the unzipped folder.
  3. Install dependencies: pip install -r requirements.txt
  4. Start locally: python start.py
  5. Open the printed Dashboard URL on your devices on the same Wi‑Fi.
  6. Set a short PIN via “Set PIN”, share the URL and PIN to unlock on the second device.
Why Local
  • No cloud traffic
  • LAN‑speed transfers
  • PIN unlock for guests
  • ZIP downloads for large folders
