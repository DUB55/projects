# Problem Set 08: CSS Architecture (BEM & SASS)

### **Goal**
Write scalable and maintainable CSS for large projects.

### **Tasks**
1. **BEM Methodology**: Explain Block, Element, Modifier (e.g., `btn`, `btn__icon`, `btn--large`).
2. **SASS Variables**: Use `$primary-color` in a `.scss` file.
3. **Nesting**: Use SASS nesting to organize styles for a navigation menu.
4. **Mixins**: Create a `@mixin` for centering elements with Flexbox.

### **Practice Drill**
Refactor a messy CSS file into a structured SASS project using partials (e.g., `_base.scss`, `_components.scss`).
