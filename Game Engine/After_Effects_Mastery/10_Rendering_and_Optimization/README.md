# Problem Set 10: Rendering and Optimization

### **Goal**
Prepare your project for final delivery.

### **Tasks**
1. **Render Queue**: Add your composition to the `Render Queue` (`Ctrl + M`).
2. **Output Modules**: Choose between ProRes (High Quality) and H.264 (Web Friendly).
3. **Adobe Media Encoder**: Learn how to send projects to Media Encoder for background rendering.
4. **Collecting Files**: Use `File` -> `Dependencies` -> `Collect Files` to package your project for another user.

### **Practice Drill**
Export your favorite "Logo Reveal" from Problem Set 02 as a high-quality .MOV file and a web-ready .MP4 file.
