# Data Analysis with Pandas

The most popular tool for data manipulation in Python.

## Concepts
- **DataFrames and Series**: The core data structures of Pandas.
- **Cleaning Data**: Handling missing values, duplicates, and outliers.
- **Filtering and Sorting**: Getting the exact data you need.
- **Grouping and Aggregating**: Summarizing data using `groupby`.

## Challenge
1. Load a CSV file into a Pandas DataFrame.
2. Find the average value of a specific column.
3. Filter the data to only show rows where a certain condition is met (e.g., "Age > 25").
