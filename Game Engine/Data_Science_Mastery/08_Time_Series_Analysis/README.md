# Problem Set 08: Time Series Analysis

### **Goal**
Analyze and forecast data that changes over time.

### **Tasks**
1. **Stationarity**: Use the Dickey-Fuller test to check if data is stationary.
2. **Seasonality**: Decompose a time series into Trend, Seasonal, and Residual components.
3. **ARIMA Models**: Understand AutoRegressive Integrated Moving Average.
4. **Exponential Smoothing**: Use Holt-Winters for forecasting.

### **Practice Drill**
Forecast "Stock Prices" or "Website Traffic" for the next 30 days using historical data.
