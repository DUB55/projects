# Problem Set 06: Network Security & TLS

### **Goal**
Understand how to protect network data from eavesdropping and tampering.

### **Tasks**
1. **Symmetric vs Asymmetric Encryption**: Explain the difference (AES vs RSA).
2. **TLS Handshake**: Describe the steps of the TLS handshake.
3. **Firewalls**: Explain the difference between stateless and stateful firewalls.
4. **VPNs**: How does a VPN tunnel protect your data on public Wi-Fi?

### **Practice Drill**
Research the "Man-in-the-Middle" attack and explain how HTTPS prevents it.
