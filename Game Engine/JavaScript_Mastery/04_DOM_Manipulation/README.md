# Problem Set 04: DOM Manipulation

### **Goal**
Interact with HTML elements using JavaScript.

### **Tasks**
1. **Selecting Elements**: Use `document.getElementById`, `querySelector`, and `querySelectorAll`.
2. **Modifying Content**: Change the `textContent` and `innerHTML` of an element.
3. **Event Listeners**: Add a `click` event listener to a button that changes the background color of the page.
4. **Creating Elements**: Use `document.createElement` and `appendChild` to add a new list item.

### **Practice Drill**
Build a simple "To-Do List" where users can add items and click them to mark as "done" (toggle a CSS class).
