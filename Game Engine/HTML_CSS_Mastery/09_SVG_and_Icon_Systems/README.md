# Problem Set 09: SVG and Icon Systems

### **Goal**
Use scalable vector graphics and icons effectively.

### **Tasks**
1. **Inline SVG**: Embed an SVG directly into HTML and change its `fill` color with CSS.
2. **Icon Fonts vs SVGs**: Explain the advantages of using SVGs over icon fonts like FontAwesome.
3. **SVG Sprites**: Learn how to use `<use xlink:href="#id">` to reuse SVG icons.
4. **Optimization**: Use a tool like SVGO to minify an SVG file.

### **Practice Drill**
Create a social media icon bar where each icon changes color and scales up on hover.
