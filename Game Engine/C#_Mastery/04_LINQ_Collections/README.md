# LINQ and Collections

Language Integrated Query - The "superpower" of C#.

## Concepts
- **Collections**: `List<T>`, `Dictionary<TKey, TValue>`, `Stack<T>`, `Queue<T>`.
- **LINQ**: Querying collections using a SQL-like syntax in C#.
  - `Where`, `Select`, `OrderBy`, `FirstOfDefault`.

## Challenge
1. Create a list of `Product` objects (Name, Price).
2. Use LINQ to find all products that cost more than $50 and sort them by name.
3. Use `Sum()` to find the total cost of all products in the list.
