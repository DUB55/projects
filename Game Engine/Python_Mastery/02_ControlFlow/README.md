# Problem Set 2: Control Flow in Python

Python uses **indentation** (whitespace) to define blocks of code, instead of the `{}` braces used in C/C++.

## Concepts Covered
- **If / Elif / Else**: Conditional logic.
- **For Loops**: Iterating over sequences (like `range()` or lists).
- **While Loops**: Repeating based on a condition.
- **Logical Operators**: `and`, `or`, `not`.

## The Exercises

### 1. Grade Calculator
Take a score (0-100) and print the letter grade (A, B, C, D, F) using `if-elif-else`.

### 2. FizzBuzz
Print numbers from 1 to 50. If divisible by 3, print "Fizz". If by 5, "Buzz". If both, "FizzBuzz".

### 3. Number Guessing
Use a `while` loop to let the user guess a secret number. Give "higher" or "lower" hints.

## Important Note
In Python, indentation is not optional. Usually, 4 spaces are used. Be consistent!
