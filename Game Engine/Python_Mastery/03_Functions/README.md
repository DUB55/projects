# Problem Set 3: Python Functions

Functions in Python are defined with the `def` keyword.

## Concepts Covered
- **Defining Functions**: `def my_function():`.
- **Arguments & Return Values**: Passing data in and out.
- **Default Arguments**: `def greet(name="User"):`.
- **Keyword Arguments**: Calling functions with `name="Alice"`.

## The Exercises

### 1. The Area Function
Write a function `calculate_area(radius)` that returns the area of a circle. (Use `3.14159` for Pi).

### 2. Multi-Return
Write a function that takes a list of numbers and returns both the `min` and the `max` in a single return statement (Python handles this using tuples automatically).

### 3. Password Validator
Write a function that checks if a string is longer than 8 characters and contains at least one number. Return `True` or `False`.

## Comparison to C
Python functions don't require you to specify return types or parameter types. This is "Duck Typing": if it walks like a duck and quacks like a duck, it's a duck!
