# Problem Set 05: Data Cleaning and Preprocessing

### **Goal**
Prepare messy, real-world data for analysis.

### **Tasks**
1. **Handling Missing Values**: Use `fillna()` and `dropna()` in Pandas.
2. **Outlier Detection**: Use the IQR (Interquartile Range) method to find outliers.
3. **Data Normalization**: Scale numerical data using Min-Max or Z-score normalization.
4. **Encoding Categorical Data**: Use One-Hot Encoding and Label Encoding.

### **Practice Drill**
Clean a "Dirty Dataset" with missing dates, duplicate rows, and inconsistent string formatting.
