# Sorting and Searching

The most common operations in computing.

## Sorting Algorithms
1. **Bubble Sort**: Simple but slow ($O(n^2)$).
2. **Merge Sort**: Fast and stable ($O(n \log n)$).
3. **Quick Sort**: Usually fastest in practice.

## Searching Algorithms
1. **Linear Search**: Check every item ($O(n)$).
2. **Binary Search**: Divide and conquer ($O(\log n)$). Requires sorted data.
