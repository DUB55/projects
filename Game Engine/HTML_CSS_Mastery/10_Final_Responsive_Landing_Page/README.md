# Problem Set 10: Final Responsive Landing Page

### **Goal**
Combine all your skills to build a professional, responsive landing page.

### **Tasks**
1. **Layout**: Use Grid for the overall structure and Flexbox for internal component alignment.
2. **Responsiveness**: Ensure the page looks great on mobile, tablet, and desktop using Media Queries.
3. **Animations**: Add subtle entrance animations using CSS keyframes.
4. **Forms**: Include a validated contact form and a footer with social icons.

### **Practice Drill**
Build a landing page for a fictional "Game Studio" including a hero section, features list, and contact form.
