# Java Collections Framework

Storing and manipulating groups of objects.

## Key Interfaces
- **List**: Ordered collection (e.g., `ArrayList`, `LinkedList`).
- **Set**: No duplicates allowed (e.g., `HashSet`, `TreeSet`).
- **Map**: Key-Value pairs (e.g., `HashMap`, `TreeMap`).

## Challenge
1. Create an `ArrayList` of Strings and sort it alphabetically.
2. Create a `HashMap` where the key is a student's name and the value is their grade.
3. Research the performance difference between an `ArrayList` and a `LinkedList`.
