# Docker and Containers

"But it works on my machine!" - Not anymore.

## Concepts
- **Image**: A blueprint of your application and its environment.
- **Container**: A running instance of an image.
- **Dockerfile**: The recipe for building an image.

## Challenge
1. Create a `Dockerfile` for a simple Python or Node.js web server.
2. Build and run the container, then access it from your browser.
