# Problem Set 10: Geometry Nodes Introduction

### **Goal**
Explore the world of procedural modeling using the Geometry Nodes system.

### **Tasks**
1. **Geometry Nodes Workspace**: Switch to the `Geometry Nodes` tab.
2. **Point Distribute**: Use the `Distribute Points on Faces` node.
3. **Instance on Points**: Use the `Instance on Points` node to place objects (like grass or rocks) on a plane.
4. **Random Value**: Use a `Random Value` node to vary the scale and rotation of your instances.

### **Practice Drill**
Create a procedural "Forest" or "Field". Start with a plane, use Geometry Nodes to distribute "Trees" (simple cylinders/cones) across the surface, and ensure they all have random sizes.
