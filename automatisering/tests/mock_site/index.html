Instructies voor Scraper Testing

Deze mock site heeft de volgende structuur voor scraper testing:

Selectors die werken:

  • Book list selector: .book-item
  • Book title selector: .book-title
  • Greeting selector: #greeting

Test workflow:

  1. Start op login.html
  2. Login met elke combinatie van e-mail/wachtwoord
  3. Je komt op dit dashboard (index.html)
  4. Klik op een boek om naar book.html te gaan
  5. In book.html: klik hoofdstukken en paragrafen

Voorbeeld configuratie voor deze mock site:

{
  "start_url": "http://localhost:8080/login.html",
  "login": {
    "use_login": true,
    "username_selector": "#email",
    "password_selector": "#password",
    "submit_selector": "button[type='submit']"
  },
  "book_selection": {
    "book_list_selector": ".book-item",
    "book_item_title_selector": ".book-title"
  }
}

Tip: Gebruik de Selector Tester in de applicatie om deze selectors te testen!
