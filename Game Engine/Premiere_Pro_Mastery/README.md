# Premiere Pro Mastery

Welcome to the **Premiere Pro Mastery** curriculum. This course is designed to turn you into a professional video editor.

## Curriculum Overview
1.  **Project Setup and Ingestion**: Organizing your workflow.
2.  **Timeline Editing Tools**: The core of cutting video.
3.  **Audio Mixing and Cleanup**: Making it sound as good as it looks.
4.  **Color Correction (Lumetri)**: Professional color grading.
5.  **Transitions and Effects**: Adding flow and style.
6.  **Titles and Essential Graphics**: Modern typography for video.
7.  **Multi-Camera Editing**: Syncing and switching multiple angles.
8.  **Keyframing Motion**: Simple animations within Premiere.
9.  **Proxy Workflows**: Editing 4K/8K footage on any computer.
10. **Export Settings and Codecs**: Delivering for YouTube, TV, and Film.

## How to use this
Each folder contains a `README.md` with a specific problem set. Complete them in order.
If you get stuck, refer to the `HELP.md` in this directory.
