# Problem Set 05: Styles and Variables

### **Goal**
Manage global design tokens like colors and typography.

### **Tasks**
1. **Color Styles**: Define a "Primary Blue" style and apply it to multiple objects.
2. **Text Styles**: Create a "Heading 1" and "Body Text" style.
3. **Color Variables**: Use the new Variables feature to create "Light" and "Dark" modes.
4. **Number Variables**: Use variables for spacing and corner radius.

### **Practice Drill**
Create a small UI kit with 4 color styles and 3 text styles. Update the "Primary Color" style and observe how your entire design changes instantly.
